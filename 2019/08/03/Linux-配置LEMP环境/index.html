Note : 此时 sudo mysql 无法继续访问mysql，我们可以通过以下命令登入：

ERROR 1819(HY000)

如果出现 ERROR 1819 (HY000): Your password does not satisfy the current policy requirements

则

sudo mysql_secure_installation

改一个强度高的密码
